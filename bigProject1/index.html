This is a website showcasing all the current Senators of the United States. If a senator's seat is up for election, clicking them will pull down the candidates. You can search for anything you want by using the search bar in the top right corner. If you would like to reorder the candidates, you can do so using the buttons on the top left. Clicking a "Sort by" button will sort the senators by your specified guideline. "Parties" will make only the selected parties show up. "State" will scroll down to the first candidate of the state you select.
